## Practica de GitHub -VictorMartínez
### Procedimiento
#### 1) Básicamente se crearon dos ramas (Branches):
              - VictorMartínez_Spring_Bootcamp2019
              - VictorMartínez_Spring_Bootcamp2019_SecondaryBranch
              
#### 2) La rama primaria creo algunos archivos, entre éstos:
              -printHelloWorld -> Archivo Java Main
              -helloWorldClass -> Una clase con algunos métodos.
              
### 3) La rama secundaria alteró algunos métodos, de hecho eliminó uno de ellos, y se procede a unir unir la rama secundaria con la primaria. 
   
### 4) Se presentarons conflictos al unir las ramas (Rebase) por lo que se editaron los archivos en conflicto:
              - helloWorldClass: Se recupera el método eliminado
              - printHelloWorld: Se da prioridad al uso del método eliminado.
              
### 5) Por último, se ignorar los archivos  basura con el archivo .gitIgnore y se procede a la subida de los archivos al repositorio remoto
